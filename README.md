Clone Time_Series file to work on the project. You can find details in [the project page](https://github.com/users/mustafa-tolu/projects/1/views/1?pane=info) how to do that.

You should relate your Pull Request with the issue which is on the repo's Pull Requests list. It is necessary to close an issue.
How will you do that:
After you opening a Pull Request for the Project, go to repo's Pull Requests tab and find yours. Then set informations which are located on the right hand side. The Development part is important because of relation with your Pull Request and the Issue that is assigned to you.
